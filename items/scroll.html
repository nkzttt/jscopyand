使い方

window.addEventListener('scroll', function () { ここに任意の処理を記述してください });
スクロールイベントは通常ではとても多くの回数が実行されます。
1ピクセル毎の処理が不要な場合や、スクロールが終わった時に処理されればいい場合は間引き（throttle / debounce）を組み合わせてください。

解説

JSコードの解説を見てみる

window.addEventListener('scroll', function () { ここに任意の処理を記述してください });
window に対して scroll というイベントを登録することで、スクロール時に任意の処理をさせることができます。
