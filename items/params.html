JSコードの解説を見てみる

URLのパラーメタ文字列をオブジェクトに変換するため、変数を定義しておきます。

location.searchには URL のパラーメタ部分の文字列が入っています。
これは ? を含むため、substring(1)で ? を除外した文字列を取得します。
取得した文字列をURLSearchParamsという関数に渡すことで、パラメータに対して様々な処理を行うことができるようになります。
今回はforEachというメソッドで、 value / key の組み合わせが引数に渡ってくる反復処理を行います。

渡ってきた value / key を最初に定義したオブジェクトの変数に入れていきます。
