JavaScriptの基本

  • どこに書けばいい？

    </body>の直上に書いてください。
    当サイトで紹介している JavaScript コードは、 HTML コードが先に読み込まれている必要があります。

  • 貼り付けるだけでいい？

    既に.jsファイルがある、または新たに作成する場合は貼り付けるだけで良いです。
    .jsを作りたくない、よくわからない場合は<script></script>というタグを作成し、その中に貼り付けてください。
